# 🚀 MERN Job Board & Task Management App

A full-stack MERN application to manage jobs and tasks efficiently.

---

## 🌐 Live Demo

* Frontend: https://your-vercel-link.vercel.app
* Backend: https://mern-job-task.onrender.com

---

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Backend: Node.js, Express.js
* Database: MongoDB Atlas
* Authentication: JWT
* Deployment: Render & Vercel

---

## ✨ Features

* 🔐 User Authentication (Login/Register)
* 📋 Task Management (Create, Update, Delete)
* 💼 Job Tracking
* 📊 Dashboard with stats
* 🌐 Fully deployed (Frontend + Backend)

## ⚙️ Installation (Local Setup)

### 1️⃣ Clone the repo

```bash
git clone https://github.com/luckysharmaaaa/mern_job_task.git
cd mern_job_task
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create `.env` file:

```
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
PORT=5000
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

### Backend:

* MONGO_URI
* JWT_SECRET
* PORT

### Frontend:

* VITE_API_URL
